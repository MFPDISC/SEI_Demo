# SEI — Sport Entertainment International
## Client Demo — GitHub Pages

### To deploy:
1. Push this folder to a GitHub repo
2. Go to repo Settings → Pages → Source: main branch / root
3. Your demo is live at: `https://yourusername.github.io/sei/`

### Demo credentials:
- Email: `demo@sei.tv`
- Password: `password123`

### To swap in a real YouTube stream:
Open `index.html`, find the `LIVE` array (~line 1030), change any `ytId` value to your YouTube Live stream ID.
